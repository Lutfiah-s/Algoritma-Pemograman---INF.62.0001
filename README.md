## `🪻MODUL 8`
### No 1 : Fungsi Rekursif
Rekursi adalah teknik pemrograman di mana sebuah fungsi memanggil dirinya sendiri. Dalam kasus perkalian, rekursi bekerja dengan prinsip berikut:

* Perkalian sebagai penjumlahan berulang: Perkalian `a * b` dapat didefinisikan sebagai penjumlahan `a` sebanyak b kali.
* Kondisi dasar: Ketika `b` mencapai 0, penjumlahan berhenti karena tidak ada lagi angka `a` yang perlu dijumlahkan.
* Kasus rekursif: Pada setiap pemanggilan fungsi, nilai `b` dikurangi 1, dan fungsi memanggil dirinya sendiri dengan nilai `b` yang baru. Proses ini berulang hingga mencapai kondisi dasar.

Cara Kerja Kode

 * `stdio.h`: Header ini menyediakan fungsi untuk input/output standar seperti printf dan scanf.
 * `math.h: Header ini menyediakan fungsi-fungsi matematika, meskipun dalam kode ini tidak digunakan secara eksplisit.
* Deklarasi Variabel:
`a`, `b`, `hasil`: Variabel bertipe float digunakan untuk menyimpan angka-angka yang dimasukkan pengguna dan hasil perhitungan. Tipe `float` dipilih karena memungkinkan representasi bilangan desimal.
* Input Angka:
Menggunakan `scanf` untuk meminta pengguna memasukkan dua angka dan menyimpannya ke dalam variabel `a` dan `b`.

## `🪻MODUL 9`
### No 1 : Pointer
* Deklarasi String:
	* `char nama[] = "BORLAND";`: Membuat sebuah array karakter bernama `nama` yang berisi string "BORLAND". Array ini akan digunakan untuk menyimpan karakter-karakter dari string tersebut.
* Hitung Panjang String:
* `int length = strlen(nama);`: Fungsi `strlen` digunakan untuk menghitung jumlah karakter dalam string `nama` dan hasilnya disimpan dalam variabel `length`.
*Loop untuk Mencetak Terbalik:
	* Inisialisasi: Loop dimulai dari `i = 1` dan akan terus berjalan hingga `i <= length`.
	* Mencetak Substring:
	 * `printf("%.*s\n", i, &nama[length - i]);`:
	   * `%.*s`: Format specifier ini digunakan untuk mencetak sebagian dari sebuah string. Angka pertama (`i`) menentukan jumlah karakter yang akan dicetak, sedangkan yang kedua (`&nama[length - i]`) adalah pointer ke awal substring yang akan dicetak.
	8 `&nama[length - i]`: Ini adalah bagian yang paling penting. `length - i` menghitung indeks karakter dari belakang. Misalnya, ketika `i = 1`, kita akan mencetak karakter terakhir. Dengan menggunakan operator `&`, kita mendapatkan alamat memori dari karakter tersebut, yang kemudian digunakan sebagai pointer untuk memulai pencetakan substring.

### No 2 : Pointer
* Include Header:
	* `stdio.h`: Header ini menyediakan fungsi-fungsi input/output standar seperti printf dan fgets.
	* `string.h`: Header ini menyediakan fungsi-fungsi untuk manipulasi string seperti strlen dan strcspn.
* Deklarasi Variabel:
	* `char kalimat[1000]`: Membuat array karakter kalimat dengan ukuran maksimum 1000 karakter untuk menyimpan kalimat yang diinputkan pengguna.
	* `int panjang, i`: Variabel panjang digunakan untuk menyimpan panjang kalimat, sedangkan i adalah variabel pengontrol loop.
*Input Kalimat:
	* `fgets(kalimat, sizeof(kalimat), stdin);`: Fungsi fgets digunakan untuk membaca sebuah baris dari input standar (keyboard) dan menyimpannya ke dalam array kalimat. Argumen sizeof(kalimat) menentukan jumlah maksimum karakter yang dapat dibaca.
	* `kalimat[strcspn(kalimat, "\n")] = '\0';`: Fungsi strcspn digunakan untuk mencari posisi karakter newline (\n) pertama dalam string. Karakter newline ini kemudian diganti dengan karakter null (\0) untuk menandai akhir string. Hal ini dilakukan karena fgets biasanya menyimpan karakter newline di akhir string.
* Hitung Panjang Kalimat:
	* `panjang = strlen(kalimat);`: Fungsi strlen digunakan untuk menghitung jumlah karakter dalam string kalimat dan hasilnya disimpan dalam variabel panjang.
* Cetak Kalimat Terbalik:
	* `for (i = panjang - 1; i >= 0; i--)`: Loop ini dimulai dari indeks terakhir string (panjang - 1) dan berjalan mundur hingga indeks 0.
	putchar(kalimat[i]);: Fungsi putchar digunakan untuk mencetak satu karakter pada setiap iterasi loop. Karakter yang dicetak adalah karakter ke-i dari string kalimat, mulai dari karakter terakhir.

### No 3 : Pointer
#### Bagian A: Menggunakan Variabel Biasa
* Deklarasi:
  	```
	int Lesley_A = 57082;: Deklarasi variabel Lesley_A dengan nilai 57082.
	int Layla_A = Lesley_A;: Deklarasi variabel Layla_A dan menginisialisasinya dengan nilai yang sama dengan Lesley_A.
	int Balmond_A = Layla_A + 1;: Deklarasi variabel Balmond_A dan menginisialisasinya dengan nilai Layla_A ditambah 1.
* Output:
```
	Nilai Layla_A akan sama dengan Lesley_A karena terjadi copy by value.
	Nilai Balmond_A akan lebih besar 1 dari Layla_A karena operasi penjumlahan.
```
#### Bagian B: Menggunakan Pointer
* Deklarasi:
  ```
	int Lesley_B = 57082;: Deklarasi variabel Lesley_B dengan nilai 57082.
	int *Layla_B = &Lesley_B;: Deklarasi pointer Layla_B dan menginisialisasinya dengan alamat memori dari variabel Lesley_B.
	int Balmond_B = *Layla_B + 1;: Deklarasi variabel Balmond_B dan menginisialisasinya dengan nilai yang diambil dari alamat yang ditunjuk oleh pointer Layla_B ditambah 1.
* Output:
  ```
	Nilai *Layla_B akan sama dengan Lesley_B karena pointer Layla_B menunjuk ke alamat memori yang sama dengan Lesley_B.
	Nilai Balmond_B akan lebih besar 1 dari Lesley_B karena operasi penjumlahan dilakukan pada nilai yang ditunjuk oleh pointer Layla_B.
